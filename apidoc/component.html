Component

Component is central resource of the system. A component is a listing of catalog item.

Resource Path: component/{id}

Resource Query Parameters:

Method Security Description /components/ Parameters Accept/Return Object
POST/PUT ADMIN - only admins may add components to the database Add a component to the database /components/
GET USER - there is no need for restriction to user access to a component Get a list of all components or a specific component's details determined by componentId /components/{componentId}/ componentId - optional
DELETE ADMIN - only admins may delete components from the database Delete a component from the database /components/{componentId}/ componentId - required
GET USER - Admin is not required to retrive tags Get the list of all tags from the tag cloud (useful for auto suggestion) /components/tags/
POST/PUT USER - Admin is not required to post or put tags Save a new tag to the tag cloud /components/tags/
DELETE ADMIN - needed in order to delete a tag from the tag cloud Delete a tag from the tag cloud perminantly /components/tags/{tagId} tagId - required
GET ADMIN - needed to provide a security layer between users and all media Get all media objects connected to components /components/media/
GET ADMIN - needed to provide a security layer betwen users and all reviews Get all reviews connected to any component /components/reviews/
GET USER - the extra security layer is not required for features Get a list of all features, or a specific feature from the features list /components/features/{featureId}/ featureId - optional
POST/PUT ADMIN - only admin may add features to the possible features list Add another feature to the feature list /components/features/
DELETE ADMIN - only admin may delete features to the possible features list Delete a feature from the feature list /components/features/{featureId}/ featureId - reqruied
GET ADMIN - needed to grab all resources connected to any component Get a list of all resources, or get a resource by id /components/resource/{resourceId} resourceId - optional
DELETE ADMIN - only admin may delete resources Delete a resource by id /components/resource/{resourceId} resourceId - required
GET USER - no extra security level is required to protect category infomration Get a list of categories, or get categories by id /components/categories/{categoryId} categoryId - optional
POST/PUT ADMIN - only admin are allowed to add categories Add a category to the list of categories /components/categories/
DELETE ADMIN - only admin are allowed to delete a category from the list of categories Delete a category from the list of categories /components/categories/{categoryId} categoryId - required
GET USER - Admin is not required to retrieve tags Get a list of tags per component or a tag specific to a component /components/{componentId}/tags/{tagId} componentId - required, tagId - optional
POST/PUT USER - Admin is not required to tag a component Add a tag specific to a component /components/{componentId}/tags/{tagId} componentId - required, tagId - required
DELETE USER - Admin is not required to un-tag a component Remove a tag specific to a component /components/{componentId}/tags/{tagId} componentId - required, tagId - required
GET User - Admin is not required to retrieve a component specific media Get the list of media speciific to a component, or get a specific media from a specific component /components/{componentId}/media/{mediaId}/ componentId - reqruied, mediaId - optional
POST/PUT ADMIN - Users will be restricted from posting media for specific components Add a piece of media to a specific component /components/{componentId}/media/{mediaId}/ componentId - reqruied, mediaId - required
DELETE ADMIN - Users will be restricted from deleting media from specific components Delete a piece of media from a specific component /components/{componentId}/media/{mediaId}/ componentId - reqruied, mediaId - required
GET User - Admin is not required to retrieve a component specific feature list Get the list of features specific to a component, or get a specific feature from a specific component /components/{componentId}/features/{featureId}/ componentId - reqruied, featureId - optional
POST/PUT ADMIN - Users will be restricted from posting features for specific components Add a feature to a specific component /components/{componentId}/features/{featureId}/ componentId - reqruied, featureId - required
DELETE ADMIN - Users will be restricted from deleting features from specific components Delete a feature from a specific component /components/{componentId}/features/{featureId}/ componentId - reqruied, featureId - required
GET User - Admin is not required to retrieve a component specific resource list Get the list of resources specific to a component, or get a specific resource from a specific component /components/{componentId}/resources/{resourceId}/ componentId - reqruied, resourceId - optional
POST/PUT ADMIN - Users will be restricted from posting resources for specific components Add a resource to a specific component /components/{componentId}/resources/{resourceId}/ componentId - reqruied, resourceId - required
DELETE ADMIN - Users will be restricted from deleting resources from specific components Delete a resource from a specific component /components/{componentId}/resources/{resourceId}/ componentId - reqruied, resourceId - required
GET User - Admin is not required to retrieve a component specific category list Get the list of categories specific to a component, or get a specific category from a specific component /components/{componentId}/categories/{categoryId}/ componentId - reqruied, categoryId - optional
POST/PUT ADMIN - Users will be restricted from posting categories for specific components Add a category to a specific component /components/{componentId}/categories/{categoryId}/ componentId - reqruied, categoryId - required
DELETE ADMIN - Users will be restricted from deleting categories from specific components Delete a category from a specific component /components/{componentId}/categories/{categoryId}/ componentId - reqruied, categoryId - required
GET USER - Admin is not required to retrieve review data for a specific component Get the list of reviews for a component, or a specific review by reviewId /components/{componentId}/reviews/{reviewId}/ componentId - required, reviewId - optional
POST/PUT USER - Admin is not required to post a review on a specific component Add a review for a specific component /components/{componentId}/reviews/ componentId - required
DELETE ADMIN - needed when the person deleting the review is not the user that gave it Delete a specific review for a specific component /components/{componentId}/reviews/{reviewId}/ componentId - required, reviewId - required
GET USER - Admin is not required to retrieve all ratings for the reviews on a certain component Get a list of all ratings from all the reviews on a certain component /components/{componentId}/reviews/rating/ componentId - required
GET USER - Admin is not required to retrieve all Get a list of all pros and cons listings for a certain component /components/{componentId}/reviews/proCons/ componentId - required
GET USER - Admin is not required to retrieve all Get a list of all of the comments for a certain component /components/{componentId}/reviews/comments/ componentId - required
GET USER - Admin is not required to get a specific review's rating Get a specific review's rating for a specific component /components/{componentId}/reviews/{reviewId}/rating/ componentId - required, reviewId - required
POST/PUT ADMIN - admin is needed only if the current user is not the same user that posted the review Post the rating of a specific review for a specific component /components/{componentId}/reviews/{reviewId}/rating/ componentId - required, reviewId - required
DELETE ADMIN - admin is needed only if the current user is not the same user that posted the review Delete the rating of a specific review for a specific component /components/{componentId}/reviews/{reviewId}/rating/ componentId - required, reviewId - required
GET USER - Admin is not required to get a specific review's proCons Get a specific pros and cons list from a specific review for a specific component /components/{componentId}/reviews/{reviewId}/proCons/ componentId - required, reviewId - required
POST/PUT ADMIN - admin is needed only if the current user is not the same user that posted the review Post the pros and cons of a specific review for a specific component /components/{componentId}/reviews/{reviewId}/proCons/ componentId - required, reviewId - required
DELETE ADMIN - admin is needed only if the current user is not the same user that posted the review Delete the pros and cons of a specific review for a specific component /components/{componentId}/reviews/{reviewId}/proCons/ componentId - required, reviewId - required
GET USER - Admin is not required to get a specific review's comments Get a list of all coments in a specific review for a specific component /components/{componentId}/reviews/{reviewId}/comments/ componentId - required, reviewId - required
POST/PUT ADMIN - admin is needed only if the current user is not the same user that posted the review Post the comments of a specific review for a specific component /components/{componentId}/reviews/{reviewId}/comments/ componentId - required, reviewId - required
DELETE ADMIN - admin is needed only if the current user is not the same user that posted the review Delete the comments of a specific review for a specific component /components/{componentId}/reviews/{reviewId}/comments/ componentId - required, reviewId - required
